AULA 01 - Introdução ao Swing e JavaFX

Por ser uma linguagem multiplataforma, o Java tem que utilizar pacotes adicionais para que os seus arquivos não sejam sobrecarregados sem necessidade. Para isso, é necessário utilizar a instrução "import" para carregar bibliotecas de pacotes adicionais. Por padrão, apenas o pacote java.lang é carregado automaticamente. Porém, caso você precise carregar bibliotecas adicionais, utilize a importação desses pacotes. Nessa aula, você vai aprender como criar uma aplicação simples utilizando Swing e JavaFX com XML. Não se esqueça de praticar bastante!
