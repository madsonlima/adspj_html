AULA 01 - Excel do zero para iniciantes

Nesta aula eu ensino passo a passo a utilizar o Microsoft Excel. Você vai aprender todo o básico necessário para se desenvolver no Excel com um conhecimento sólido e estruturado. Existem muitos tutoriais de Excel no youtube, porém aqui você irá aprender de forma gradativa como usar o Excel, para que você logo deixe de ser um iniciante e passe a dominar o Excel como profissional. Se você vai fazer um concurso, se precisa utilizar o Excel no trabalho ou na escola, este é o curso certo para você. Aqui nunca irei lhe oferecer nada para comprar, todo o conhecimento sobre Excel será oferecido apenas com o intuito de lhe ajudar a adquirir uma nova habilidade, incluindo a anatomia de uma planilha, como inserir dados, como fazer com que seus dados tenham boa aparência, tornando a visualização mais fácil. Este tutorial foi feito usando o Excel do Office 365, mas é aplicável a versões mais antigas do Excel e a versões mais recentes também. No vídeo você encontrará também uma dica de como conseguir o excel gratuitamente direto da Microsoft.

AULA 02 - Dicas e truques de nível intermediário no Excel

Dando continuidade à nossa aula de Excel anterior, vamos ver como copiar a formatação de uma célula para outras. Como fazer formatação condicional no Excel, como criar fórmulas, somar, tirar a média, e indicar o valor máximo e mínimo de uma sequência de valores. Além disso vamos ver como funciona a função condicional SE no Excel. Por último conheça uma das mais poderosas funções do Excel, a função de autocompletar ou autopreenchimento. Com esta função você cria a sua tabela do Excel em poucos minutos e sem precisar ficar digitando várias vezes as mesmas fórmulas

AULA 03 - Tabela Dinâmica e Gráfico Dinâmico

Neste vídeo eu mostro passo a passo como elaborar uma tabela dinâmica e um gráfico dinâmico. Se você não conhece este recurso do Excel, você está perdendo uma ferramenta muito poderosa para você manipular suas planilhas e gráficos. O vídeo foi feito em forma de aula, na qual eu dou muitas outras informações de utilização do Excel, por exemplo, como ocultar ou mostrar colunas no Excel.
